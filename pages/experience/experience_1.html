Satellite Firmware Development Intern

This internship involved developing embedded software for data communication between a nanosatellite payload board and its central computer. The project aimed to ensure robust performance for upcoming radiation tests, focusing on real-time communication and automated testing processes.

  • Concepts: Radiation Testing, Real-Time Communication

Results and Conclusions

  • Developed embedded software for data communication between a nanosatellite payload board and its central computer, ensuring robust performance for upcoming radiation tests.
  • Designed and optimized real-time communication protocols (I2C, UART) in C for STM32 microcontrollers, achieving a 99.85% reduction in execution time.
  • Automated testing processes with a custom Bash script interface for algorithm selection and error tracking.
  • Managed project versions with SVN and integrated automated builds using CMake.
  • Used GDB for debugging and OpenOCD for programming embedded systems.
