Project Details

Description

This project involves simulating a 2-PAM (Pulse Amplitude Modulation) digital transmission chain using MATLAB. It focuses on modeling and implementing both analog and digital filters for effective signal processing. Additionally, the project includes estimating the Power Spectral Density (PSD) using FFT and IFFT techniques to measure and analyze the spectral response of the filters, ensuring optimal performance in the transmission chain.
