Project Details

Real-Time Control System for Autonomous Vehicle

Description

This project involves developing a real-time control system for an autonomous vehicle, focusing on implementing precise control loops and dynamic equations for accurate navigation. The project integrates motor and steering control with FreeRTOS for real-time task management and leverages the NXP FRDM-KL25Z microcontroller programmed with MCU Expresso.

  • Context: Project for the course "Contrôle-commande temps réel des systèmes cyber-physiques" in the Embedded Systems and Connected Objects master's program.
  • Local: Institut Polytechnique de Grenoble (INP Grenoble), Grenoble, France
  • Duration: 5 months
  • Team Size: 2 people
  • Timeline: Sep 2024 - Jan 2025

Technologies Used

  • Development Boards: NXP FRDM-KL25Z
  • Programming Languages: C
  • Tools: FreeRTOS, MCU Expresso
  • Concepts: Real-Time Control, System Modeling

Tasks

  • Developing control loops and dynamic equations for real-time precision in autonomous vehicle navigation.
  • Integrating motor and steering control with FreeRTOS for effective real-time task scheduling.
  • Programming the NXP FRDM-KL25Z microcontroller using MCU Expresso for control system implementation.
  • Testing and validating the system to ensure responsiveness and accuracy in navigation tasks.
