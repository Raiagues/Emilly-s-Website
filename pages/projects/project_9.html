Project Details

Embedded Software for a CubeSat

Description

This project involves developing embedded software for a CubeSat, focusing on the implementation of communication protocols for real-time data exchange between the satellite’s microcontroller and the ground station. The project also includes the development of autonomous embedded software for efficient task management, incorporating attitude control algorithms to maintain optimal satellite orientation during operation.

  • Context: Project for the course "FabLab" in the Embedded Systems and Connected Objects master's program.
  • Local: Institut Polytechnique de Grenoble (INP Grenoble), Grenoble, France
  • Duration: 5 months
  • Team Size: 2 people
  • Timeline: Sep 2023 - Jan 2024

Technologies Used

  • Development Boards: ESP32, ESP8266
  • Programming Languages: C
  • Tools: Arduino IDE
  • Concepts: UDP, Real-Time Control, FSM Design

Tasks

  • Implemented communication protocols for real-time data exchange between the CubeSat’s microcontroller and the ground station.
  • Developed autonomous embedded software for effective task management in satellite operations.
  • Incorporated attitude control algorithms to ensure optimal orientation and stability of the CubeSat.
  • Tested and validated communication and control systems to ensure reliability during missions.
