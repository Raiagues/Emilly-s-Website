Project Details

Real-Time Control System of a Magnetic Suspension

Description

This project involves designing a real-time control system for a magnetic suspension system. The approach includes system modeling using physical equations and linearizing around the equilibrium point, followed by discretization and stability analysis. MATLAB simulations validate the model, and control is implemented with a state feedback LQR regulator and a state observer based on a Kalman filter, using Simulink for real-time simulation.

  • Context: Project for the course "Contrôle-commande temps réel des systèmes cyber-physiques" in the Embedded Systems and Connected Objects master's program.
  • Local: Institut Polytechnique de Grenoble (INP Grenoble), Grenoble, France
  • Duration: 1 month
  • Team Size: Individual
  • Timeline: Oct 2024 - Nov 2024

Technologies Used

  • Programming Languages: MATLAB
  • Tools: Simulink
  • Concepts: System Modeling, LQR, Kalman Filter, Real-Time Control

Tasks

  • Modeling the magnetic suspension system using physical equations and linearizing around the equilibrium point.
  • Performing model discretization and stability analysis, validated through MATLAB simulations.
  • Implementing state feedback control with an LQR regulator and a state observer using a Kalman filter in Simulink.
  • Simulating the control system in real-time to ensure precise system response and stability.
