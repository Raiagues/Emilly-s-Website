Java Compiler for a Subset of an Advanced Language

Description

This project focuses on illustrating essential aspects of software engineering through the development of a complex application: a Java-based compiler for a subset of an advanced programming language. Key aspects include compliance with requirements, software design, validation and verification techniques, and a structured quality process. The project aims to provide a comprehensive understanding of programming language concepts while adhering to rigorous software engineering principles.
