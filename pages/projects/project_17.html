Project Details

Description

This project aims to support aerospace engineering students through teaching and outreach activities, helping them develop skills beyond the classroom while increasing community engagement with academic events. EP Aero is Brazil's first pilot school for Aerospace Engineering.

  • Context: Educational outreach project.
  • Local: Santa Maria, Brazil
  • Duration: 1 year and 3 months
  • Team Size: 15 person
  • Timeline: Nov 2022 - Jan 2024

Technologies Used

  • Programming Languages: JavaScript, HTML, CSS
  • Tools: SonarQube, Scalingo, Node, React, Git, CI/CD

Tasks

  • Responsible for structuring internal projects, contributing to the selection process of new members, allocating members to activities, delegating and tracking tasks, and initiating partnerships with aerospace professionals, institutions, and projects.
  • Involved in planning aerospace mini-courses for students and the external community, reaching over 200 participants.
  • Developed an interactive website to manage aerospace events and workshops, utilizing JavaScript frameworks to enhance event registration and the presentation of activities.
  • Integrated SonarQube in the CI/CD pipeline to ensure continuous code quality analysis, detect bugs, and enhance security.
