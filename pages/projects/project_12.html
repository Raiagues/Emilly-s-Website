Project Details

Description

This project involves developing a Spell Checker with efficient data structures and algorithms for optimized word search and correction. Key components include implementing tries, hash tables, and balanced binary trees, and comparing the performance of various search and sorting algorithms to determine the most efficient approach for spell-checking.

  • Context: Project for the course "Algorithmique et programmation" in the Embedded Systems and Connected Objects master's program.
  • Local: Institut Polytechnique de Grenoble (INP Grenoble), Grenoble, France
  • Duration: 4 months
  • Team Size: 2 people
  • Timeline: Oct 2023 - Jan 2024

Technologies Used

  • Programming Languages: C
  • Tools: Git, MakeFile, Valgrind
  • Concepts: Sequential Search, Sorting Algorithms, Tries, Hash Tables, Balanced Binary Trees

Tasks

  • Implementation of advanced data structures, including tries, hash tables, and balanced binary trees.
  • Compared efficiency of search and sorting algorithms for spell-checking tasks.
  • Organized project structure using Makefiles for seamless compilation and execution.
  • Performance and complexity analysis to optimize processing speed and memory usage.
