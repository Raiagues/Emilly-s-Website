Project Details

Description

This project focuses on the validation and evaluation of the CVA6 processor upgraded to a 128-bit RISC-V architecture. The aim is to ensure robust performance of the new processor design while automating the validation process to enhance efficiency. This entails a detailed analysis of microarchitecture through reverse engineering and meticulous documentation review. The project also involves developing and automating tests in Assembly and C to validate functionality and optimize FPGA performance.

  • Context: Project for the course "Projet de Systèmes Embarqués" in the Embedded Systems and Connected Objects master's program.
  • Local: Institut Polytechnique de Grenoble (INP Grenoble), Grenoble, France
  • Duration: 5 months
  • Team Size: 2 people
  • Timeline: Sep 2024 - Jan 2025

Technologies Used

  • Programming Languages: C, Assembly, Verilog
  • Tools: GCC, GDB, Vivado, Verilator
  • Concepts: Microarchitecture Analysis, Validation, Low-Level Software

Tasks

  • Currently upgrading the CVA6 processor to 128-bit architecture, aimed at enhancing performance capabilities.
  • Developing an automated testing framework for efficient validation processes.
  • In-depth microarchitecture analysis in progress to identify key optimizations for FPGA synthesis and improve area and performance.
  • Creating comprehensive documentation to support future enhancements and maintenance.
