Image Filter Development on System on Chip

This project involves developing an image filter on a System on Chip (SoC) platform using C++ and SystemC. The project includes implementing a communication bus for efficient data transfer between SoC modules and programming a finite state machine (FSM) in Verilog to manage pixel-level filtering processes. Additionally, a custom testbench is created to generate test images and analyze waveform outputs for performance evaluation.

  • Context: Project for the course "Modélisation transactionnelle des systèmes sur puces" in the Embedded Systems and Connected Objects master's program.
  • Local: Institut Polytechnique de Grenoble (INP Grenoble), Grenoble, France
  • Duration: 4 months
  • Team Size: 2 people
  • Timeline: Sep 2024 - Dec 2024

Technologies Used

  • Programming Languages: C++, SystemC, Verilog
  • Tools: SystemC, MakeFile
  • Concepts: FSM Design, Data Bus Communication

Tasks

  • Implementing an image filter in C++ and SystemC with a communication bus for data transfer between SoC modules.
  • Programming a finite state machine (FSM) in Verilog to control pixel-level filtering.
  • Developing a testbench in C++ to generate signals and analyze waveforms for filter validation.
  • Documenting design and validation processes to support future enhancements and optimizations.
